UNIT 1 · INTRODUCTION

What is Statistics?

Statistics
Methods used to intelligently collect, summarize, analyze, present, and interpret data for the purposes of decision making.

Descriptive Statistics

Statistical methods used to organize, summarize, and present data in a convenient and informative way.

Activities

  • Collecting data (surveys)
  • Presenting data (tables, graphs)
  • Summarizing data (sample mean, etc.)

Inferential Statistics

The process of making an estimate, prediction, or decision about a population based on sample data.

Activities

  • Estimation (estimating population mean)
  • Hypothesis testing
  • Making decisions / forecasts

Key Vocabulary

Population (N)
The set of ALL elements of interest in a particular study.
Sample (n)
A subset (portion) of the population.
Census
Collecting data from the entire population. Often impossible, expensive, or time-consuming.
Parameter
A numerical measure that describes a characteristic of a population. Value is almost always unknown — must be estimated.
Statistic
A numerical measure that describes a characteristic of a sample. Used as an estimate of the unknown parameter.
Simple Random Sample
Each element of the population has an equal chance of being selected.
💡
Remember: Parameter → Population. Statistic → Sample. The sample mean estimates the population mean. All statistics are estimates.

Types of Data & Variables

Qualitative (Categorical) Data

Labels or names used to identify an attribute of each element. Often referred to as categorical data.

  • Uses nominal or ordinal scale
  • Can be nonnumeric or numeric
  • Display with bar charts or pie charts

Examples

Marital status, eye colour, preferred banking mode, job stress level (very/somewhat/none)

Quantitative (Numerical) Data

Numeric values that represent measured or counted characteristics.

  • Discrete: Counted items (number of children, defects per hour)
  • Continuous: Measured characteristics (weight, voltage, temperature)
  • Display with histograms

Examples

Daily max temperature, vehicles owned, income, stock price

Quantitative Data: Building a Frequency Distribution

Steps to construct a frequency distribution for grouped numerical data:

1
Find the Range — Range = Max − Min
2
Choose number of classes (k) — Use Sturges' Rule as a guide: k ≈ 1 + 3.322 × log₁₀(n)
3
Compute class width (c) — c = Range ÷ k (round up)
4
Determine class limits — Start at or below the minimum value
5
Count observations — Tally data into each class and fill in the table

Histogram

A bar chart for quantitative data. Bars are adjacent (no gaps) and represent class intervals on the x-axis.

Use for:

  • Visualizing distribution shape
  • Identifying skewness, symmetry

Frequency Polygon

Line graph connecting the class midpoints at their frequencies. Midpoint = (Lower + Upper) / 2.

Midpoints for temp example:

15, 25, 35, 45, 55

Ogive (Cumulative % Curve)

Plots the upper class boundary against the cumulative percentage. Also called the "Less Than Ogive".

  • Starts at (lower boundary, 0%)
  • Ends at (upper boundary of last class, 100%)

Stem-and-Leaf Display

Splits each data value into a stem (leading digit/s) and a leaf (last digit). Shows the distribution shape while preserving actual values.

Example: 12 → Stem: 1 | Leaf: 2 35 → Stem: 3 | Leaf: 5

DESCRIPTIVE STATISTICS

Measures of Central Tendency

📌
Central tendency is the extent to which all data values group around a central value. Three main measures: Mean, Median, Mode.

Mean (Arithmetic Mean)

The most common measure. Sum of all values divided by number of values.

Sample Mean
x̄ = (Σxᵢ) / n where n = sample size
Population Mean
μ = (Σxᵢ) / N where N = population size

Properties

  • Unique — only one mean per dataset
  • Affected by outliers/extreme values — use median instead if data is skewed
  • Uses all values in the calculation

Example

Data: 2, 3, 5, 1, 4, 3, 2, 4 → Mean = 24/8 = 3

Median

The middle value when data is arranged in ascending order. Splits the data 50/50.

Median Position Formula
Position = (n + 1) / 2 If n is ODD → median is the middle value If n is EVEN → median is the average of the two middle values

Properties

  • Not affected by extreme values (outliers)
  • Unique — one median per dataset
  • Preferred over the mean for skewed data (e.g. home prices)

Example

Ordered data: 12, 23, 24, 27, 34, 38, 40, 40, 42 → Position = 5 → Median = 34

Mode

The most frequently occurring value in a dataset.

Properties

  • Not affected by extreme values
  • Can be used for qualitative OR quantitative data
  • There may be no mode (all values unique)
  • There may be multiple modes (bimodal, multimodal)

Example

7, 11, 14.25, 15, 15, 15.5, 19, 19, 19, 19, 19, 21 → Mode = 19 (occurs 4 times)

Which measure to use?

SituationUseReason
Normal/symmetric distributionMeanUses all data, most informative
Skewed data or outliers presentMedianNot distorted by extreme values
Categorical data / most common valueModeOnly measure for qualitative data

Classic Example — House Prices

Prices: $2,000,000 / $500,000 / $300,000 / $100,000 / $100,000

Mean = $3,000,000 / 5 = $600,000 ← distorted by the outlier Median = $300,000 ← more representative Mode = $100,000 ← most frequent

DESCRIPTIVE STATISTICS

Measures of Variation (Dispersion)

📌
Variation is the amount of dispersion or scattering of values. Two datasets can have the same mean but very different spreads.

Range

Simplest measure of variation.

Range = X_largest − X_smallest

Disadvantages

  • Ignores how data are distributed in between
  • Sensitive to outliers — one extreme value changes the range dramatically

Quartiles & Interquartile Range (IQR)

Quartiles split ranked data into 4 equal segments of 25% each.

Quartile Positions (ranked data)
Q1 position = 0.25 × (n + 1) Q2 position = 0.50 × (n + 1) ← Median Q3 position = 0.75 × (n + 1)
IQR Formula
IQR = Q3 − Q1 Eliminates top and bottom 25%, measures middle 50% spread.

Example

Ranked data: 11, 12, 13, 16, 16, 17, 18, 21, 22 (n = 9)

Q1 position = 0.25 × 10 = 2.5 → average of 2nd & 3rd = (12+13)/2 = 12.5 Q3 position = 0.75 × 10 = 7.5 → average of 7th & 8th = (18+21)/2 = 19.5 IQR = 19.5 − 12.5 = 7

Variance & Standard Deviation

The most commonly used measures. Each value is used in the calculation; values far from the mean receive extra weight because deviations are squared.

Sample Variance
s² = Σ(xᵢ − x̄)² / (n − 1) Note: divide by (n−1), not n
Sample Standard Deviation
s = √s²
Population Variance
σ² = Σ(xᵢ − μ)² / N Note: divide by N
Population Standard Deviation
σ = √σ²

Steps to calculate variance

1
Calculate the mean (x̄)
2
Subtract the mean from each value: (xᵢ − x̄)
3
Square each deviation: (xᵢ − x̄)²
4
Sum all squared deviations: Σ(xᵢ − x̄)²
5
Divide by (n − 1) for sample, N for population

Coefficient of Variation (CV)

Measures relative variation — always expressed as a percentage. Useful to compare datasets measured in different units or with different means.

CV = (s / x̄) × 100%

Example — Comparing Two Stocks

StockMean PriceStd DevCVConclusion
Stock A$50$510%More variable relative to price
Stock B$100$55%Less variable relative to price

Both stocks have the same standard deviation ($5), but Stock B is less variable relative to its price.

Shape & Distribution Rules

The Empirical Rule (Bell-Shaped / Normal Distribution)

Applies when data has a bell-shaped (symmetric) distribution.

68%
μ ± 1σ
of values fall within 1 standard deviation of the mean
95%
μ ± 2σ
of values fall within 2 standard deviations of the mean
99.7%
μ ± 3σ
of values fall within 3 standard deviations of the mean

Chebyshev's Theorem

Works for any distribution (regardless of shape). For k > 1:

At least (1 − 1/k²) × 100% of values fall within k standard deviations of the mean
k valueIntervalAt least % of data
k = 1.5μ ± 1.5σ1 − 1/1.5² = 55.6%
k = 2μ ± 2σ1 − 1/4 = 75%
k = 3μ ± 3σ1 − 1/9 = 89%
⚡
Key difference: Empirical Rule = only for bell-shaped data (exact percentages). Chebyshev = any distribution (minimum percentages).

Advanced Topics

Geometric Mean

Used to measure the rate of change of a variable over time, especially investment returns. More accurate than arithmetic mean for growth rates.

Geometric Mean Rate of Return: xg = [(1+x₁)(1+x₂)...(1+xn)]^(1/n) − 1

Example

Investment: $100,000 → $150,000 (year 1, +50%) → $180,000 (year 2, +20%)

Arithmetic mean = (50% + 20%) / 2 = 35% ← MISLEADING Geometric mean = √(1.50 × 1.20) − 1 = √1.80 − 1 ≈ 34.16% ← ACCURATE

Weighted Mean / Grouped Data Approximations

When data is already grouped into K classes (frequencies f₁, f₂,… and midpoints m₁, m₂,…):

Mean from Grouped Data
x̄ = Σ(fᵢ × mᵢ) / n where n = Σfᵢ
Variance from Grouped Data
s² = Σfᵢ(mᵢ − x̄)² / (n − 1)

Covariance & Correlation Coefficient

Covariance
Measures the strength and direction of the linear relationship between two variables. No standardized scale.
Cov(X,Y) > 0 → move same direction Cov(X,Y) < 0 → move opposite directions Cov(X,Y) = 0 → independent
Correlation Coefficient (r)
Measures the relative strength of the linear relationship. Always between −1 and +1.
r = −1 → perfect negative linear r = 0 → no linear relationship r = +1 → perfect positive linear
💡
The correlation coefficient is unit free — it doesn't depend on the units of measurement. Covariance is NOT unit free.

Five-Number Summary & Box-and-Whisker Plot

Five-Number Summary: Minimum | Q1 | Median (Q2) | Q3 | Maximum IQR = Q3 − Q1 Lower fence = Q1 − 1.5 × IQR Upper fence = Q3 + 1.5 × IQR Values outside fences are OUTLIERS

Key Concepts & Definitions

🚨
Your lecturer said: NB!! Know the concepts and definitions. These are the most likely definition questions. Know each term cold.
Data
Observed values (observations) on a variable. Raw data = data that has not been summarized or organized.
Variable
A characteristic that changes or varies over time and/or for different individuals or objects under consideration.
Discrete Variable
Can only take specific (countable) values. E.g. number of children, number of defects.
Continuous Variable
Can take any value within a range (measured). E.g. weight, height, temperature.
Descriptive Measure
A single number used to describe a set of data. Includes measures of central tendency and measures of dispersion.
Central Tendency
The extent to which all data values group around a central value.
Variation / Dispersion
The amount of scattering or spread in the data values.
Frequency Distribution
A summary table where data is grouped into class intervals, each with a corresponding frequency.
Class Interval / Class Width (c)
The range of values in a single class. c = Range ÷ k. Classes have lower and upper class limits.
Class Midpoint
The middle value of a class interval. Midpoint = (Lower limit + Upper limit) / 2. Used in frequency polygons and grouped data calculations.
Relative Frequency
The proportion of observations in a class. = Frequency / Total n. All relative frequencies sum to 1.
Ogive (Less Than Ogive)
A cumulative frequency graph. Plots upper class boundaries against cumulative percentages.
Outlier
An extreme value that is far from the rest of the data. Outliers distort the mean and range but not the median.
Skewness
A measure of asymmetry in a distribution. Positive skew = tail to right. Negative skew = tail to left.
IQR (Interquartile Range)
IQR = Q3 − Q1. Measures the spread of the middle 50% of the data. Resistant to outliers.
Coefficient of Variation (CV)
CV = (s / x̄) × 100%. Measures relative variation as a percentage. Used to compare variability across different units or scales.
Empirical Rule
For bell-shaped distributions: 68% within 1σ, 95% within 2σ, 99.7% within 3σ of the mean.
Chebyshev's Theorem
For ANY distribution: at least (1 − 1/k²) of values fall within k standard deviations of the mean (k > 1).
Correlation Coefficient (r)
Measures the relative strength of the linear relationship between two variables. Ranges from −1 to +1. Unit free.
Covariance
Measures the strength and direction of the linear relationship between two variables. NOT unit free.
